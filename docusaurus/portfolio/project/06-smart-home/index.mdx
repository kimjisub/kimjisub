## 프로젝트 소개

`ESP32`칩을 이용하여 릴레이 모듈을 제어해, 집에 있는 각종 전등들을 원격으로 제어할 수 있도록 서버와 제어 모듈을 개발했습니다. 단순한 원격 제어가 아닌, Google Smart Home Api를 연동하여, 인공지능 스피커 Google Home 또는 Google Assistance 앱을 활용하여 범용성 있게 활용할 수 있도록 하였습니다.

## 사용한 기술

- `actions-on-google`을 이용하여 Google Smart Home Api 연결
- `node.js`를 이용하여 Google Smart 장치를 연결해주는 서버 구성
- `ESP32`를 이용한 네트워크 소켓 통신
- 납땜을 하여 장치 소형화
