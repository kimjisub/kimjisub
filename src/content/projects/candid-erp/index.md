# Candid ERP

스타트업 채용 컨설팅 Core Engine

---

## 🎯 프로젝트 개요

Candid의 채용 컨설팅 업무를 지원하는 자체 ERP 시스템. Excel, Slack, 머릿속에 흩어져 있던 데이터를 단일 시스템으로 통합하여 **Data-Driven 의사결정**이 가능한 조직으로 전환.

---

## 🔧 기술적 도전 & 해결

### 도메인 모델링
- **문제**: 채용 컨설팅 도메인을 어떻게 데이터 구조로 설계할 것인가
- **해결**: 회사/포지션/후보자/진행단계/컨설턴트를 유기적으로 연결하는 관계형 모델 설계

### ETL (Extract, Transform, Load)
- **문제**: 기존 Excel/Slack에 분산된 데이터를 어떻게 마이그레이션할 것인가
- **해결**: 데이터 정합성 검증 후 단계적 마이그레이션

### 도메인 이해
- **문제**: "어떤 정보를 수집해야 하는지" 정의 — 기술이 아니라 비즈니스 문제
- **해결**: 컨설턴트들과 협업하여 워크플로우 분석 및 데이터 요구사항 도출

### 복잡한 빌링 구조
- **문제**: 채용 수수료 구조가 복잡 (성공 보수, 단계별 청구 등)
- **해결**: 유연한 빌링 모델 설계 및 시스템화

## 💡 UX 혁신

### Table 기반 UI
- Notion/Airtable 스타일의 직관적인 테이블 인터페이스
- 컨설턴트들이 기존 Excel에서 자연스럽게 전환 가능

### 고급 필터 기능
- 복잡한 조건 조합으로 후보자/회사 검색
- 컨설턴트 워크플로우에 최적화
