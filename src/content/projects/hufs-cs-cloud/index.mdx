한국외대 컴퓨터공학과 학생들에게 개인 Ubuntu 인스턴스를 제공하는 교내 클라우드 서비스입니다. 수업에서 리눅스 환경이 필요할 때마다 각자 VM을 설치하는 번거로움을 없애기 위해 시작했습니다.

## 배경

컴공과 수업 중 리눅스 환경에서 실습해야 하는 과목이 많았는데, 학생마다 로컬에 VM을 설치하거나 WSL을 세팅하는 과정에서 환경 문제가 자주 발생했습니다. 학과 차원에서 표준화된 Ubuntu 인스턴스를 제공하면 수업 진행이 훨씬 수월해질 거라 판단했습니다.

## 주요 작업

- Kubernetes 클러스터를 구축하여 학생별 격리된 Ubuntu 인스턴스를 동적으로 생성하고 관리하는 시스템을 설계했습니다.
- 각 학생이 웹 브라우저를 통해 자신만의 리눅스 환경에 접속할 수 있도록 구현했습니다.
- Shell 스크립트를 활용하여 인스턴스 생성/삭제를 자동화했습니다.
- 학과 학생 수에 맞게 리소스 스케일링이 가능한 구조로 설계했습니다.
