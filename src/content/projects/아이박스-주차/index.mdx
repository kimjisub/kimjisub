주차장 운영에 필요한 입출차 관리, 정산, 할인, 모니터링을 하나로 통합하는 전산 시스템입니다. 외주 계약으로 참여하여 모바일 앱과 웹 관리자 페이지, 백엔드를 개발했습니다.

## 주차 정산 로직

다양한 주차 정산 시나리오를 지원하도록 설계했습니다.

- 내부 주차장 시간 기반 과금
- 정액 할인권 (고정 금액 차감)
- 정률 할인권 (비율 기반 차감)
- 결제 바코드를 이용한 정액/정률 할인권 연동

## 성능 개선

AWS CloudWatch 로그를 분석하여 병목 구간을 파악하고, 쿼리 최적화 및 인프라 튜닝을 통해 응답 시간을 개선했습니다.

## 주요 작업

- React Native 기반 크로스플랫폼 모바일 앱을 개발하고, Fastlane으로 배포를 자동화했습니다.
- React PWA로 관리자 페이지를 구현했습니다.
- Node.js + MongoDB로 백엔드를 개발했습니다.
- Docker, AWS S3, GitHub Actions CI/CD로 인프라를 구축했습니다.
