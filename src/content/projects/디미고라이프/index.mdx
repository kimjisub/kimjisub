## 배경

한국디지털미디어고등학교에서 전교생이 기숙사 생활을 했습니다. 주말 잔류, 세탁, 학생 지도일지 등 기숙사 생활에 필요한 문서들을 수기로 관리하고 있었고, 신청서 누락과 중복이 종종 발생했습니다. 이 문제를 해결하기 위해 기숙사 전산화 시스템을 기획하고 개발했습니다.

## 주요 기능

- 주말 잔류, 지도일지, 급식 신청 등 학교 생활에 필요한 기능을 온라인으로 제공했습니다.
- 선생님들이 신청 내용을 표와 그래프로 조회하고 관리할 수 있도록 구성했습니다.
- 전산화된 데이터를 기존 엑셀 형식으로 내보낼 수 있게 하여 새로운 시스템에 쉽게 적응할 수 있도록 했습니다.
- Socket.io를 활용하여 실시간 현황 모니터링 기능을 개발했습니다.

## 주요 작업

- Express.js, MySQL, Sequelize를 활용하여 REST API를 개발했습니다.
- Swagger 문서화를 통해 프론트 팀에게 API 문서를 제공했습니다.
- 급식 신청의 경우 비용에 직결되는 문제였기 때문에 수차례의 보안 테스트를 거쳤습니다.

## 성과

- 600여 명의 전교생과 선생님들이 사용하는 서비스로 운영했습니다.
