# UniPad

1,000만+ 다운로드를 달성한 런치패드 기반의 연주형 리듬게임

![image](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/20ebab5d-17df-41dd-9a8e-bdcba091096d/unipad-A4.jpg)

## 핵심 성과

- **1,000만+ 다운로드** (Google Play)
- 누적 사용 수 **2억 4천만 회**
- 평균 동접자 **2,000명**
- 음악게임 부문 **1위**, 전체게임 **45위** 기록
- 현재 **오픈소스로 전환**되어 커뮤니티 주도 운영 중

---

## 🎯 핵심 가치: 플랫폼으로서의 설계

단순 앱이 아니라, **사용자가 직접 콘텐츠(유니팩)를 만들고 공유하는 순환 구조**를 설계했습니다.

- 콘텐츠 생산자 → 소비자 → 다시 생산자로 전환되는 생태계
- 유니팩 제작자, 연주자, 시청자가 유기적으로 연결

---

## 🔧 기술적 도전 & 해결

### 레이턴시 문제
- **문제**: 모바일에서 버튼을 누르자마자 소리가 나야 함. Java AudioTrack으로는 지연 발생
- **해결**: NDK 활용 저지연 오디오 재생 구현
- **결과**: 실시간 연주가 가능한 수준의 레이턴시 확보

### 콘텐츠 제작 진입장벽
- **문제**: 유니팩 제작이 어려워 콘텐츠 공급 부족
- **해결**: 자동 음악 슬라이싱 도구 개발
- **결과**: 누구나 쉽게 유니팩 제작 가능 → 콘텐츠 폭발적 증가

### 성능 최적화
- Task 최적화하여 **초당 200회 이상의 LED Emulation** 처리
- MIDI 장치 연동을 위한 자체 드라이버 인터페이스 개발
- AAC Room 이용한 로컬 데이터베이스 구축
- LiveData, ViewModel, DataBinding 등 아키텍처 적용

- **17개국어 번역** — 글로벌 시장 공략

### 커뮤니티 운영
- 네이버 카페, Facebook, Discord 등 **다채널 커뮤니티 직접 운영**

### 바이럴 성장
- 런치패드 유튜버들이 UniPad 사용 → **자연스러운 노출**
- 티비플, 해외 커뮤니티 등을 통한 입소문
