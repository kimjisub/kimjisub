한국외대 교내 세미나실/스터디룸의 예약 현황을 조회하고 예약할 수 있는 웹 시스템. 기존에 수동으로 관리하던 예약 프로세스를 자동화했다.

<Image src={img_content_1_png} alt="" />

## 주요 기능

- 예약 현황 캘린더: 세미나실별 실시간 예약 현황 조회
- 온라인 예약: 원하는 시간대를 선택하여 바로 예약
- 본인인증: 비밀번호 재설정을 위한 학교 메일 기반 본인인증
- 예약 알림 및 관리: 예약 확인, 변경, 취소 기능

## 기술 스택

- 프론트엔드: React 기반 SPA로 예약 캘린더 및 관리 UI 구현
- 백엔드: Spring Boot로 예약 로직, 인증, 관리자 기능 개발
- 팀 프로젝트로 프론트엔드와 백엔드를 분담하여 개발
