한화비전에서 운영하는 아파트 커뮤니티 및 주차 관리 솔루션 모바일 앱입니다. 외주 계약 형태로 참여하여 React Native 기반 크로스플랫폼 앱 개발을 담당했습니다.

<Image src={img_content_1_png} alt="" />

## 맡은 업무

- 한화비전 아파트 커뮤니티 및 주차 관리 앱을 React Native로 개발했습니다.
- Android(Kotlin)와 iOS(Swift) 네이티브 모듈을 연동했습니다.
- 디자인 팀과 주기적으로 회의하고 피드백을 반영하여 UI/UX를 개선했습니다.
- Firebase를 활용하여 푸시 알림 및 실시간 데이터 연동을 구현했습니다.

## 기술적 특징

React Native로 Android/iOS를 동시에 대응하면서도 각 플랫폼의 네이티브 기능(Kotlin, Swift)을 활용했습니다. TypeScript를 도입하여 코드 안정성을 확보하고, Firebase를 활용하여 푸시 알림, 인증, 실시간 데이터 처리를 구현했습니다.
