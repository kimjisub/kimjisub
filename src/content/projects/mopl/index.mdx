한화비전에서 운영하는 아파트 커뮤니티 및 주차 관리 솔루션 모바일 앱. 외주 계약 형태로 참여하여 React Native 기반 크로스플랫폼 앱 개발을 담당했다.

<Image src={img_content_1_png} alt="" />

## 맡은 업무

- 한화비전 아파트 커뮤니티 및 주차 관리 앱 React Native 개발
- Android(Kotlin)와 iOS(Swift) 네이티브 모듈 연동
- 디자인 팀과 주기적인 회의 및 피드백을 통한 UI/UX 개선
- Firebase 푸시 알림 및 실시간 데이터 연동

## 기술적 특징

- React Native로 Android/iOS 동시 대응하면서도 각 플랫폼 네이티브 기능(Kotlin, Swift)을 활용
- TypeScript 도입으로 코드 안정성 확보
- Firebase를 활용한 푸시 알림, 인증, 실시간 데이터 처리
