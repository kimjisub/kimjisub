## 배경

하나의 LLM만 사용하면 그 모델의 편향이나 한계에 갇히기 쉽습니다. 같은 질문에 대해 여러 모델의 관점을 비교하고, 서로를 비판하게 하면 더 균형 잡힌 인사이트를 얻을 수 있다고 생각하여 개발했습니다.

## 주요 기능

- Multi-Model Debate: 동일한 질문을 여러 AI 모델에 동시에 전송하고 응답을 나란히 비교합니다.
- Cross-Examination: 각 모델의 응답을 다른 모델에게 피드백으로 전달하여 진정한 다관점 토론을 진행합니다.
- Peer Ranking: 각 참가자가 다른 모델들의 기여도를 평가하고 순위를 매깁니다.
- Custom Roles: 시스템 프롬프트로 커스텀 페르소나를 정의하여 각 모델이 특정 관점에서 주장하도록 설정할 수 있습니다.
- No API Keys Required: 이미 로그인한 브라우저 탭을 통해 동작하므로 별도 API 키가 불필요합니다.

## 지원 Provider

- ChatGPT (chatgpt.com)
- Gemini (gemini.google.com)
- Claude (claude.ai)

## 기술 스택

React 19와 Zustand로 UI를 구현하고, Vite + @crxjs/vite-plugin으로 빌드했습니다. Chrome Extension Manifest V3의 Side Panel API를 활용하여 브라우저 사이드 패널에서 동작합니다.
