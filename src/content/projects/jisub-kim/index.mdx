지금 보고 계신 이 포트폴리오 웹사이트입니다. 프로젝트, 경력, 기술 스택, 블로그를 한 곳에서 보여주기 위해 직접 기획하고 개발했습니다.

## 주요 작업

- Next.js 16 (App Router) + React 19 + TypeScript로 전체 웹사이트를 구축했습니다.
- Tailwind CSS + Radix UI Themes를 활용하여 다크모드 지원 UI를 구현했습니다.
- MDX 기반 콘텐츠 관리 시스템을 구축하여, Notion에서 관리하던 방식에서 코드와 콘텐츠를 같은 레포에서 관리하는 방식으로 전환했습니다.
- D3.js로 프로젝트-기술-경력 간의 관계를 포스 그래프로 시각화하는 기능을 개발했습니다.
- Claude API를 연동하여 방문자가 직접 대화할 수 있는 AI 터미널 기능을 구현했습니다.
- PostgreSQL(Supabase) + Prisma ORM으로 방문자 채팅 기록과 통계를 관리합니다.
