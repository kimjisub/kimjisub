지금 보고 계신 이 포트폴리오 웹사이트. 프로젝트, 경력, 기술 스택, 블로그를 한 곳에서 보여주는 개인 사이트다.

## 기술 스택

- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS + Radix UI Themes
- MDX 기반 콘텐츠 관리
- PostgreSQL (Supabase) + Prisma ORM
- D3.js 기반 프로젝트/기술 관계 그래프 시각화
- AI 채팅 터미널 (Claude API 연동)

## 특징

콘텐츠를 Notion에서 관리하던 방식에서 MDX 파일 기반으로 전환하여, 코드와 콘텐츠를 같은 레포에서 관리할 수 있게 했다. 프로젝트-기술-경력 간의 관계를 D3 포스 그래프로 시각화하는 기능과, 방문자가 직접 대화할 수 있는 AI 터미널 기능을 넣었다.
