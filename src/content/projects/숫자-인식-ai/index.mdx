외부 라이브러리(TensorFlow, PyTorch 등) 없이 Java만으로 신경망을 밑바닥부터 구현하여 손글씨 숫자를 인식하는 프로젝트. 고등학교 시절 AI/딥러닝의 원리를 직접 이해하고 싶어서 시작했다.

## 구현 내용

행렬 연산, 활성화 함수, 역전파(backpropagation) 알고리즘을 전부 직접 구현했다. 입력 이미지를 픽셀 단위로 읽어서 다층 퍼셉트론(MLP)에 통과시키고, 0~9 중 어떤 숫자인지 분류한다.

## 배경

딥러닝 프레임워크를 가져다 쓰면 모델을 쉽게 만들 수 있지만, 내부에서 실제로 무슨 일이 일어나는지 이해하기 어렵다. 경사 하강법이 가중치를 어떻게 조정하는지, 역전파가 어떤 순서로 기울기를 전달하는지를 코드로 직접 작성하면서 신경망의 학습 과정을 체감할 수 있었다.
