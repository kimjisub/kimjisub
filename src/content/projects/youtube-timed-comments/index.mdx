## 배경

유튜브 댓글 중 특정 시간을 태그한 댓글이 많지만, 영상을 보면서 해당 시점의 댓글을 확인하기가 불편했습니다. 또한 긴 영상에서 하이라이트 구간을 빠르게 찾고 싶은 수요가 있었습니다. 이를 해결하기 위해 시간 태그 댓글을 분석하는 Chrome Extension을 개발했습니다.

## 주요 기능

- 실시간 댓글 동기화: 영상 시청 중 현재 재생 시점에 해당하는 시간 태그 댓글을 자동으로 표시했습니다.
- 하이라이트 분석: 시간 태그 댓글의 분포를 분석하여 영상 타임라인에 그래프로 시각화했습니다. 댓글이 집중된 구간을 하이라이트로 판별하여 사용자가 원하는 지점에 빠르게 접근할 수 있도록 했습니다.

## 주요 작업

- 유튜브 댓글에서 시간 태그를 파싱하고 시간순으로 정렬하는 로직을 구현했습니다.
- 영상 재생 시점과 댓글을 동기화하는 기능을 개발했습니다.
- 시간대별 댓글 밀도를 타임라인 그래프로 시각화했습니다.
