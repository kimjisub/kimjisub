# Vplate

마케팅 영상 제작 플랫폼

### 회사 소개

더브이플래닛은 광고영상 자동제작 및 송출 플랫폼 **Vplate**를 서비스 합니다.

- *Vplate*는 영상에 대한 전문지식이 없는 사람들도, 몇번의 Tap만으로 멋진 디자인과 훌륭한 퍼포먼스를 만들어내는 광고영상을 제작할 수 있는 자동제작 유틸리티를 제공합니다.
### 맡은 업무

- 안드로이드 앱 개발
- 백엔드 API 서버 개발
- After Effect Script를 이용한 자체 렌더링 엔진 개발
- 렌더링 서버 병렬 처리 시스템 개발
