더브이플래닛에서 서비스하는 광고영상 자동제작 및 송출 플랫폼 [Vplate](https://vplate.io/). 영상 전문지식 없이도 몇 번의 탭만으로 고품질 마케팅 영상을 만들 수 있는 서비스다.

## 맡은 업무

- 안드로이드 앱 개발
- 백엔드 API 서버 개발
- After Effects Script를 활용한 자체 렌더링 엔진 개발
- 렌더링 서버 병렬 처리 시스템 구축

## 기술적 특징

사용자가 입력한 영상, 이미지, 텍스트를 템플릿에 맞춰 자동으로 합성하는 과정에서 Adobe After Effects의 스크립트 API를 활용했다. 렌더링 작업이 오래 걸리기 때문에 여러 서버에 작업을 분산하는 병렬 처리 시스템을 구축하여 처리 속도를 개선했다.
