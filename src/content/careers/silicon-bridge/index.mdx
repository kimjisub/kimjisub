아파트 차량 관제 및 커뮤니티 플랫폼 Eyevacs를 개발하는 Silicon Bridge에서 풀스택 엔지니어로 근무했습니다. 약 4년 7개월간 재직하며(중간 휴식기 1년 7개월 포함) 모바일 앱, 백엔드, 인프라 전반을 담당했습니다.

## 맡은 업무

- React Native 기반 모바일 앱을 개발했습니다. (82 pages, 120 APIs)
- NestJS 백엔드 API 서버를 개발하고 MSA 아키텍처를 설계했습니다.
- AWS(ECS, RDS, S3, CloudWatch) 기반 인프라를 구축하고 운영했습니다.
- 클라우드에서 Bare Metal로의 인프라 마이그레이션을 수행했습니다.
- 미국 현장 납품을 위한 i18n, 타임존 시스템을 구축했습니다.
- IoT 장비(ESP32) 펌웨어 TLS 인증서 갱신 전략을 설계했습니다.
- 디자인 시스템 컴포넌트 36개를 구축하고 자체 컴포넌트 라이브러리를 개발했습니다.
