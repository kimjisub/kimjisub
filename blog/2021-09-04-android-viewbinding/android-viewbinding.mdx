`kotlin-android-extension` 플러그인이 `deprecated` 되면서 대안으로 사용할 수 있는 `viewBinding`에 대해서 설명한다.

<!--truncate-->

## 프로젝트에 viewBinding 적용하기

```kotlin title="build.gradle [Module]"
android{
	buildFeatures {
		viewBinding true
		// dataBinding true
	}
}
```

## Activity

Activity의 경우 다음과 같다

```kotlin title="MainActivity.kt"
class MainActivity : Activity() {
	private lateinit var b: ActivityMainBinding

	override fun onCreate(savedInstanceState: Bundle?) {
		super.onCreate(savedInstanceState)
		b = ActivityMainBinding.inflate(layoutInflater)
		setContentView(b.root)
	}
}
```

## Custom View

직접 만든 레이아웃과 뷰는 다음과 같다. xml 파일의 이름은 `view_custom.xml`이라고 가정한다.

```kotlin title="CustomView.kt"
class CustomView
@JvmOverloads constructor(
	context: Context,
	attrs: AttributeSet? = null,
	defStyleAttr: Int = 0
) : RelativeLayout(context, attrs, defStyleAttr) {
	private val b: ViewCustomBinding =
		ViewCustomBinding.inflate(LayoutInflater.from(context), this, true)
}
```
